## Задания

### Задание 1: Установка LAMP-стека

**1.1. Подготовка системы**
```bash
# Обновление системы
apt-get update
apt-get upgrade -y

# Проверка доступности репозиториев
apt-cache search apache2
apt-cache search php
apt-cache search mariadb
```

**1.2. Установка Apache HTTP Server**
```bash
# Установка Apache
apt-get install apache2 -y

# Проверка установки
apache2 -v
systemctl status apache2

# Проверка работы (должен быть доступен по http://localhost)
systemctl start apache2
systemctl enable apache2
```

**1.3. Установка PHP**
```bash
# Установка PHP и основных модулей
apt-get install php php-mysql php-cli php-common php-gd php-mbstring php-xml php-curl -y

# Проверка установки
php -v
php -m | grep -E "(mysql|gd|mbstring|xml|curl)"
```

**1.4. Установка MySQL/MariaDB**
```bash
# Установка MariaDB
apt-get install mariadb-server mariadb-client -y

# Проверка установки
mysql --version
systemctl status mariadb

# Первоначальная настройка безопасности
mysql_secure_installation
```

**1.5. Проверка работы LAMP**
```bash
# Создание тестового файла
echo "<?php phpinfo(); ?>" > /var/www/html/info.php

# Перезагрузка Apache
systemctl restart apache2

# Проверка в браузере: http://ваш_сервер/info.php
# После проверки удалите файл!
rm /var/www/html/info.php
```

```bash

<VirtualHost *:80>
```

### Задание 4: Создание PHP-приложения

**4.1. Создание конфигурационного файла**
```bash
# Создание файла конфигурации
cat > /var/www/example.com/public_html/config.php << 'EOF'
<?php
// Конфигурация базы данных
define('DB_HOST', 'localhost');
define('DB_NAME', 'webapp');
define('DB_USER', 'webuser');
define('DB_PASS', 'SecurePassword123!');
define('DB_CHARSET', 'utf8mb4');

// Создание подключения через PDO
function getDBConnection() {
    try {
        $dsn = "mysql:host=" . DB_HOST . ";dbname=" . DB_NAME . ";charset=" . DB_CHARSET;
        $options = [
            PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION,
            PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
            PDO::ATTR_EMULATE_PREPARES => false,
        ];
        return new PDO($dsn, DB_USER, DB_PASS, $options);
    } catch (PDOException $e) {
        throw new PDOException($e->getMessage(), (int)$e->getCode());
    }
}
?>
EOF

# Назначение прав (только для чтения владельцем)
chmod 640 /var/www/example.com/public_html/config.php
chown www-data:www-data /var/www/example.com/public_html/config.php
```

**4.2. Создание страницы добавления пользователя**
```bash
cat > /var/www/example.com/public_html/add_user.php << 'EOF'
<?php
require_once 'config.php';

$message = '';

if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    $username = trim($_POST['username'] ?? '');
    $email = trim($_POST['email'] ?? '');
    $password = $_POST['password'] ?? '';
    
    if (!empty($username) && !empty($email) && !empty($password)) {
        try {
            $pdo = getDBConnection();
            $stmt = $pdo->prepare("INSERT INTO users (username, email, password_hash) VALUES (?, ?, ?)");
            $password_hash = password_hash($password, PASSWORD_DEFAULT);
            $stmt->execute([$username, $email, $password_hash]);
            $message = "User created successfully!";
        } catch (PDOException $e) {
            $message = "Error: " . $e->getMessage();
        }
    } else {
        $message = "Please fill all fields!";
    }
}
?>
<!DOCTYPE html>
<html>
<head>
    <title>Add User</title>
</head>
<body>
    <h1>Add New User</h1>
    <?php if ($message): ?>
        <p><?php echo htmlspecialchars($message); ?></p>
    <?php endif; ?>
    <form method="POST">
        <label>Username: <input type="text" name="username" required></label><br>
        <label>Email: <input type="email" name="email" required></label><br>
        <label>Password: <input type="password" name="password" required></label><br>
        <button type="submit">Add User</button>
    </form>
    <p><a href="index.php">Back to Home</a> | <a href="users.php">View Users</a></p>
</body>
</html>
EOF
```

**4.3. Создание страницы просмотра пользователей**

```bash
            <tr>
                <th>ID</th>
                <th>Username</th>
                <th>Email</th>
                <th>Created</th>
            </tr>
            <?php foreach ($users as $user): ?>
                <tr>
                    <td><?php echo htmlspecialchars($user['id']); ?></td>
                    <td><?php echo htmlspecialchars($user['username']); ?></td>
                    <td><?php echo htmlspecialchars($user['email']); ?></td>
                    <td><?php echo htmlspecialchars($user['created_at']); ?></td>
                </tr>
            <?php endforeach; ?>
        </table>
    <?php else: ?>
        <p>No users found.</p>
    <?php endif; ?>
    <p><a href="index.php">Back to Home</a> | <a href="add_user.php">Add User</a></p>
</body>
</html>
EOF
```

### Задание 5: Настройка безопасности

**5.1. Создание .htaccess файла**
```bash
cat > /var/www/example.com/public_html/.htaccess << 'EOF'
# Отключение листинга директорий
Options -Indexes

# Защита от XSS
<IfModule mod_headers.c>
    Header set X-XSS-Protection "1; mode=block"
    Header set X-Content-Type-Options "nosniff"
    Header set X-Frame-Options "SAMEORIGIN"
</IfModule>

# Ограничение загрузки файлов
<IfModule mod_php.c>
    php_value upload_max_filesize 10M
    php_value post_max_size 10M
</IfModule>

# Кэширование
<IfModule mod_expires.c>
    ExpiresActive On
    ExpiresByType text/css "access plus 1 month"
    ExpiresByType application/javascript "access plus 1 month"
    ExpiresByType image/jpeg "access plus 1 year"
    ExpiresByType image/png "access plus 1 year"
</IfModule>
EOF

# Назначение прав
chmod 644 /var/www/example.com/public_html/.htaccess
```

**5.2. Настройка базовой аутентификации для админ-панели**
```bash
# Создание директории админ-панели
mkdir -p /var/www/example.com/public_html/admin

# Создание .htpasswd файла

EOF

chmod 644 /var/www/example.com/public_html/admin/.htaccess
```

### Задание 6: Настройка SSL (опционально)

**6.1. Создание самоподписанного SSL-сертификата**
```bash
# Создание директории для SSL
mkdir -p /etc/apache2/ssl

# Генерация самоподписанного сертификата
openssl req -x509 -nodes -days 365 -newkey rsa:2048 \
    -keyout /etc/apache2/ssl/example.com.key \
    -out /etc/apache2/ssl/example.com.crt \
    -subj "/C=RU/ST=Sverdlovsk/L=Yekaterinburg/O=Test/CN=example.com"

# Назначение прав
chmod 600 /etc/apache2/ssl/example.com.key
chmod 644 /etc/apache2/ssl/example.com.crt
```

**6.2. Настройка виртуального хоста с SSL**
```bash
# Включение модуля SSL
a2enmod ssl

# Создание конфигурации для HTTPS
cat > /etc/apache2/sites-available/example.com-ssl.conf << 'EOF'
<VirtualHost *:443>
    ServerAdmin admin@example.com
    ServerName example.com
    ServerAlias www.example.com
    DocumentRoot /var/www/example.com/public_html
    
    SSLEngine on
    SSLCertificateFile /etc/apache2/ssl/example.com.crt
    SSLCertificateKeyFile /etc/apache2/ssl/example.com.key
    
    <Directory /var/www/example.com/public_html>
        Options -Indexes +FollowSymLinks
        AllowOverride All
        Require all granted
    </Directory>
    
    ErrorLog ${APACHE_LOG_DIR}/example.com-ssl-error.log
    CustomLog ${APACHE_LOG_DIR}/example.com-ssl-access.log combined
</VirtualHost>
EOF

# Включение сайта
a2ensite example.com-ssl.conf

# Перезагрузка Apache
systemctl reload apache2
```

**6.3. Настройка перенаправления HTTP на HTTPS**
```bash
# Обновление HTTP виртуального хоста
cat > /etc/apache2/sites-available/example.com.conf << 'EOF'
```
